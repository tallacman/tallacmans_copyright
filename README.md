# Tallacmans Copyright 
Tallacmans Copyright for Concrete CMS keeps your site’s copyright notice accurate, readable, and on-brand—without touching it every January. Built for Concrete CMS 9 and modern PHP environments, it displays a clean, professional notice that updates the current year automatically and lets you fine-tune every detail from the block editor.

Set your copyright holder, optional start year (for ranges like 2010–2026), and typography in one place. Choose from your curated font presets or enter a custom CSS font stack, then adjust weight, size, and color with a live preview before you publish. Block padding controls help the notice sit comfortably in headers, footers, and sidebars.

- Features
Automatic year updates — The notice always shows the current year, so your copyright stays accurate without manual edits.
- Optional year range — Add a start year to display a span (e.g. 2010–2026) that reflects how long your content has been protected.
- Copyright holder field — Show your company, brand, or name clearly alongside the copyright symbol.
- Full typography controls — Pick a font preset or custom stack, plus weight (thin through black), size, and text color.
- Live preview — See font, weight, size, and color changes in the block editor before saving.
- Flexible spacing — Top and bottom padding help the notice fit naturally in any layout.
- Modern Concrete CMS integration — A refreshed edit interface, Concrete 9 standards, and output designed for today’s themes.
